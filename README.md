# 阿里云公共云 · 市场洞察 Demo 看板

一个基于 **Streamlit + Plotly** 构建的交互式可视化看板，用于展示 2025Q2 阿里云及主要竞争对手的市场份额、AI 云发展态势、广告/营销渠道，以及华为云 / 腾讯云 / 百度智能云等对手的对比分析。  
本项目仅用于学习与面试演示，数据来源于公开渠道（Synergy Research、Omdia、Canalys、Similarweb 等）。

---

## ✨ 功能概览

- **① 市场总览**
  - 全球云市场 Treemap
  - 中国云市场份额（环形图 + 条形图）
  - 中国 AI 云份额条形图
- **② 机遇与挑战**
  - AI / 非AI 驱动机遇散点图
  - 挑战与风险矩阵（Impact × Likelihood）
  - 风险缓解策略表格
- **③ 阿里云广告与营销**
  - 海外站 / 中国站流量结构（饼图）
  - 策略要点总结
- **④ 竞争对手分析**
  - **雷达图分面**：展示阿里云与华为云 / 腾讯云 / 百度智能云的能力画像
  - **气泡图**：份额 × 增速 × 利润 proxy 三维分析
